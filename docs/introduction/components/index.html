IDX Components

This page contains an overview of the libraries and components utilized within IDX.

Application Libraries#

JavaScript/TypeScript libraries used within your application.

IDX#

js-idx is the core library used to interact with the IDX protocol.

Developer Utilities#

Libraries used during development with IDX.

IDX Tools#

idx-tools provides development utilities for building applications using the IDX protocol.

IDX CLI#

idx-cli provides various commands to interact with DIDs and IDX documents, as introduced in this getting started guide. Note, the idx-cli requires the Ceramic CLI below.

Dependencies#

IDX depends on Ceramic for storing documents and DIDs for identifiers and authentication, so these libraries will be needed in various places. When required, they will be referenced in-line in this documentation.

Ceramic#

Ceramic CLI#

Ceramic CLI can be used to run a local Ceramic node, which is needed for IDX.

DID Providers#

3ID DID Provider#

3ID DID Provider is a DID Provider used to instantiate and authenticate 3IDs, allowing them to create, sign, and update documents on Ceramic.

3ID Connect#

3ID Connect provides an iframe-based version of 3ID DID Provider that can be used by Web applications. 3ID Connect allows users to rely on their existing Ethereum wallet for signing. Support for more blockchains is coming soon.
