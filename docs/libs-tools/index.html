IDX Tools library

Constants#

schemas#

A record of the Schemas, keyed by IDX schema name:

  • BasicProfile: see Basic Profile CIP
  • CryptoAccountLinks: see Crypto Account Links CIP
  • Definition (CIP to be defined)
  • IdentityIndex see Identity Index CIP

Returns Record<IDXSchemaName, Schema>

signedDefinitions#

A record of signed Definitions, keyed by IDX definition name:

Returns Record<IDXDefinitionName, DagJWSResult>

signedSchemas#

A record of the signed schemas

Returns Record<IDXSchemaName, DagJWSResult>

publishedDefinitions#

A record of published Definitions, keyed by IDX definition name:

Returns Record<IDXDefinitionName, DocID>

publishedSchemas#

A record of the published schemas

Returns Record<IDXSchemaName, DocID>

API#

validateSchema#

Checks if a JSON Schema is valid and secure according to Ajv's secure schema

Arguments

  1. schema: Schema

Returns boolean

publishDoc#

Creates or updates a document defined by the PublishDoc interface

Type parameters

  1. T = unknown

Arguments

  1. ceramic: CeramicApi
  2. doc: PublishDoc<T>

Returns Promise<DocID>

createDefinition#

Creates and publishes a new Definition

Arguments

  1. ceramic: CeramicApi
  2. definition: Definition

Returns Promise<DocID>

updateDefinition#

Similar to publishDoc for an existing DefinitionDoc

Arguments

  1. ceramic: CeramicApi
  2. doc: DefinitionDoc

Returns Promise<boolean> whether the definition contents have changed

publishSchema#

Similar to publishDoc for a SchemaDoc, with additional validation using validateSchema

Arguments

  1. ceramic: CeramicApi
  2. doc: SchemaDoc

Returns Promise<DocID>

publishIDXConfig#

Publishes the signed Definitions and Schemas provided by IDX

Arguments

  1. ceramic: CeramicApi

Returns Promise<IDXPublishedConfig>
